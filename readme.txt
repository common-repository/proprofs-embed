== Installation ==

1. Download ProProfs Embed Widget from the WordPress plugin admin panel in the plugin directory. 
2. Go to your WordPress Administration -> Plugins and activate ProProfs LMS WordPress plugin.
3. Add shortcode `[ProProfs src="https://proprofs.com/" width="100%" height="500"]` to page or post.
4. Get the src code from [ProProfs](https://www.proprofs.com/).

= Shortcode Params: =
* **src** - Only work with ProProfs URL/domain. Get the src code from [ProProfs](https://www.proprofs.com/)
* **width** - Width is in percentage or in pixels - [ProProfs width="100%"] or [ProProfs width="600"], default width="100%"
* **height** - Height is in percentage  or in pixels - [ProProfs height="100%"] or [ProProfs width="600"] , default height="500"
* **scrolling** - With or without the scrollbar - [ProProfs scrolling="no"] or  [ProProfs scrolling="yes"] 
* **frameborder** - With or without the frame border - [ProProfs frameborder="0"]
